# Smart POS & Full Inventory Management App

A production-ready Flutter application for small/medium shops with complete POS, inventory, customer ledger, reports, offline support, and Google Drive backup.

## Features (In Progress)
- User Authentication (Login/Signup)
- Product & Inventory Management
- POS Billing System
- Customer Management & Ledger
- Reports
- Offline Mode with SQLite
- Auto Sync
- Backup & Restore (Local + Google Drive)

## Technology Stack
- Flutter (Dart)
- Backend: Firebase (chosen for auth + future Firestore)
- Local DB: SQLite (sqflite)
- State Management: Provider / Riverpod (to be decided)
